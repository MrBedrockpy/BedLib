## BedLib

BedLib - это новая переработанная версия моей библиотеки для комфортного и быстрого создания плагинов для Paper-совместимых ядер для Minecraft. Я полностья переписал системы для конфигураций и менеджмента данными. 

## Why you should use this library?

1. Моя библиотека стандартизирует любой плагин, который использует эту библиотеку, до единой структуры в проекте плагина.
